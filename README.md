### Main Service
- **사용자의 개발 에러를 해결하기 위한 질문/답변 게시판**
    - GPT 기반 자동 생성 질문/답변 제공
    - 마크다운 에디터를 통한 편리한 질문/답변 UX
    - ELK를 적용한 텍스트 및 기술 태그 기반 검색 기능
- 에러 해결을 위한 채팅 및 화면공유 서비스
- 소셜 및 랭킹 기능
